# Exotic Ceylon Exports Website

A professional and modern website for Exotic Ceylon Exports, highlighting premium Ceylon Tea and Spices.

## Setup Instructions

This is a static website built with pure HTML, CSS and JavaScript. No build tools are required.

1. **Open the Website**: Simply double-click `index.html` to view the site in your browser.
2. **Edit Content**: Open `index.html` in any text editor (like VS Code or Notepad) to update text, product details, or contact information.
3. **Replace Images**: 
   - The images used are high-quality placeholders from Unsplash.
   - To use your own product photos, place your images in the `assets` folder.
   - Update the `src` attribute of the `<img>` tags in `index.html` to point to your new images (e.g., `src="assets/my-tea-image.jpg"`).

## Features

- **Responsive Design**: Looks great on mobile, tablet, and desktop.
- **Glassmorphism Header**: Modern, transparent navigation bar.
- **Smooth Animations**: Elements fade in as you scroll.
- **SEO Optimized**: Includes basic meta tags for search engines.
- **Contact Form**: Professional layout ready for backend integration.

## Customization

To change the color theme, edit the `:root` variables in `styles.css`:

```css
:root {
    --primary-color: #046307; /* Emerald Green */
    --accent-color: #D4AF37; /* Gold */
}
```

Enjoy your new website!
